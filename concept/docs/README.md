# Documentation

Welcome to the Taskify application documentation. This folder contains comprehensive guides for understanding, configuring, deploying, and developing the application.

## Documentation Structure

- **[ARCHITECTURE.md](ARCHITECTURE.md)** - System architecture, design decisions, Azure services used, and component interactions
- **[CONFIGURATION.md](CONFIGURATION.md)** - Service configuration, environment variables, and setup instructions
- **[DEPLOYMENT.md](DEPLOYMENT.md)** - Deployment procedures, infrastructure as code, and runbooks
- **[DEVELOPMENT.md](DEVELOPMENT.md)** - Development environment setup, coding standards, and local development guide
- **[GitHub-MBR.md](GitHub-MBR.md)** - Monthly/quarterly business review documentation

## Quick Start

1. **Getting Started?** → Start with [DEVELOPMENT.md](DEVELOPMENT.md) to set up your local environment
2. **Understanding the System?** → Read [ARCHITECTURE.md](ARCHITECTURE.md) for system design
3. **Deploying to Azure?** → Follow [DEPLOYMENT.md](DEPLOYMENT.md)
4. **Need Configuration Details?** → Check [CONFIGURATION.md](CONFIGURATION.md)
